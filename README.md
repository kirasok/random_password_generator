# Random Password Generator

A Flutter app that generator different passwords

It have ability to use different characters:  
- lowercase (a-z)  
- uppercase (A-Z)  
- symbols (!, #, " etc)  
- numbers (0-9)

Also it can exclude:  
- similar (like 0-O, 1-l and others)  
- ambiguous (~,;:.{}\<\>\[\]\(\)/\'`)
